# DreamersJoy Floral Studio

Cinematic website for DreamersJoy Floral Studio — a boutique floral design studio in Wyckoff, Bergen County, NJ.

## Brand Identity

- **Identity:** Modern Minimal Luxury + Editorial Contemporary
- **Founder:** Sarah De Jesus
- **Location:** Wyckoff, New Jersey
- **Tagline:** Refined Floral Design for Intimate Gatherings

## Stack

- Vanilla HTML/CSS/JS (single-page cinematic site)
- GSAP 3.12.5 + ScrollTrigger (scroll-driven animations)
- Google Fonts (Cormorant Garamond + Jost)
- ElevenLabs ConvAI Widget (AI voice agent)
- Deployed on Vercel

## Structure

```
site/
  index.html          # Main cinematic website
  assets/
    frames/           # Hero video frames (scroll-driven)
competitive-report.html   # 20-competitor market analysis
proposal.html             # Site build proposal
project.json              # Project configuration
```
